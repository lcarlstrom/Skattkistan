Skattkistan är en lösenordshanterare skapt för att underlätta lösenordsgeneration samt hantering för användare av alla slag.
Programmet har en enkel grafisk interface där användare kan definiera längden av önskat lösenord, generera detta med ett knapptryck och
sedan visa detta genererade lösenord.

Skattkistan körs genom att starta skriptet varpå användaren får ta del av ett grafiskt interface med en input-låda och en generations-knapp.
I input-lådan bör ett integer värde matas in baserat på önskad lösenordslängd.
Efter input har fyllts klickar användaren på generations-knappen och får lösenord i den högre spalten.
Dessa lösenord syns default som asterisker med en knapp vid sidan för att visa/dölja lösenordet.
